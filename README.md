 ##### 具体思路（知识点）：  
    1. 异步加载相册图片；  
    2. 自定义相册文件夹；  
    3. 支持单选、多选（最多9张）图片；  
    4. ItemTouchHelper实现拖拽、排序、删除；  
    5. 对Canvas画布操作、变换，结合属性动画，实现图片的放大、缩小等；
